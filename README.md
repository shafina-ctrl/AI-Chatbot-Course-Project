# AI-Chatbot-Course-Project
Chatbot untuk menjawab soalan kursus AI.

## Problem & Persona
**Problem:** Pelajar kursus AI sering perlukan bantuan cepat untuk faham konsep asas tanpa tunggu kelas atau pensyarah.  
**Persona:** Seorang pelajar universiti yang ambil kursus AI dan mahu chatbot sebagai tutor ringkas.

## Prototype
Chatbot dibina menggunakan Python + OpenAI API.  
Boleh dijalankan di terminal atau melalui Streamlit untuk interface web.
